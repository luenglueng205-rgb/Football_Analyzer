# 2026 AI-Native Quantitative Sports Betting System
> The Digital Life Cluster Edition

## 架构说明 (Architecture Overview)
本项目已经历了三次大规模重构，最终确立为 **100% AI-Native** 的多智能体操作系统（Agentic OS）。

### 核心目录结构
*   `core/`: Agentic OS 核心引擎，包括灵魂独白 (`soul_monologue`)、长记忆海马体 (`hippocampus`)、顶级路由 (`grandmaster_router`)、动态防幻觉风控 (`hallucination_guard`)。
*   `core/digital_life/`: 数字生命集群的高阶模块（多模态视觉感知、达尔文繁衍引擎、Web3经济主权）。
*   `skills/`: AI 可自主调用的确定性工具集。包括最硬核的数学函数（泊松分布、凯利公式、平博去水）和 OpenClaw 的多模态视觉 MCP。
*   `global_knowledge_base/`: 系统的物理记忆区。包含微型 NLP 模型 (`.onnx`)、长短期记忆 JSON、回测沙盒。

## 运行与演化
系统已被设计为全自动自治模式。
1. 启动数字生命集群：`python3 core/docs/superpowers/tests/test_digital_life.py`
2. 启动 AI 自主投研：`python3 core/ai_quant_researcher.py`
