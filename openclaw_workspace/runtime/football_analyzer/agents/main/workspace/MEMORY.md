## 记忆系统配置

### 情景记忆 (Episodic Memory)
- 存储位置: `data/memory/episodic.json`
- 最大记录: 1000条
- 保留时间: 30天
- 内容: 历史投注案例、胜负记录

### 语义记忆 (Semantic Memory)
- 存储位置: `data/memory/semantic.json`
- 内容: 联赛知识、球队特征、赔率规律
- 自动更新: 是

### 程序记忆 (Procedural Memory)
- 存储位置: `data/memory/procedural.json`
- 内容: 投注流程、分析步骤、风控规则
- 学习机制: 自动优化
